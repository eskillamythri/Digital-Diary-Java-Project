# Digital-Diary-Java-Project

A Java Swing based desktop application used to create, save, open, edit, and delete personal notes securely.

## Features
- Password Protection
- Create Notes
- Save Notes
- Open Notes
- Delete Notes
- Simple GUI

## Technologies Used
- Java
- Swing
- File Handling

## How to Run
1. Open project in NetBeans or Eclipse
2. Run Login.java
3. Enter password: admin123

```text
src
├── Login.java
├── DiaryHome.java

```
